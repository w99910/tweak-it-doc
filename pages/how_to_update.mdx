# Suggested ways to update the extension

We host our extension on the Chrome Web Store and Firefox Add-ons.

import { Callout } from 'nextra/components';

<Callout type="info">
  Both platforms automatically update the extension to the latest version.
</Callout>

### Manual Update
However, if you want to manually update the extension, follow the steps below.

<Callout type="warning">
  Please proceed the following steps with caution as it cannot be undone.
</Callout>

1. Open the extension and go to settings tab.
2. In order to save your custom styles and applied settings, turn on `Store current settings and sync across your devices` settings. ( Please note that if you are free user, you can't use this settings.)
3. Remove the extension.
4. Reinstall the extension from related store (Chrome Web Store or Firefox Add-ons).
