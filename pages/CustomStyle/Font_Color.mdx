# Font Color

You can create your favourite font color and apply it either to specific website or as global style.

You can configure font colors of Header elements and Paragraph elements by tweaking in Header and Paragraph configuration. 

<Callout type="info">
  In order to reset the font color, leave the input box as blank.
</Callout>
