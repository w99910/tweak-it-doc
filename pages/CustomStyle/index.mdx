# Overview

### What is a style?

Style refers to a collection of settings that control the appearance of elements, including

There are three types of configuration in a style.

- General: General Configuration for all layout. 
- Header: In order to overwrite style for the Header Elements, this configuration can be used.
- Paragraph: In order to overwrite style for the paragraph Elements, you can tweak this paragraph configuration.
