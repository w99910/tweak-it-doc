# Font Family

Currently, we only support Google Fonts. In order to apply Google Font, you need to provide the name of Google Font.

- Browse your favourite Google font at [Google Fonts](https://fonts.google.com/)
- Enter the font name in the input box.

<Callout type="info">
  In order to reset the font family, leave the input box as blank.
</Callout>
